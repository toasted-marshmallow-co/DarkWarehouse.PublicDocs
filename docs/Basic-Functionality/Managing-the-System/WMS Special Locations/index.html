WMS Special Locations 📍

WMS Special Locations are unique locations in the Dark Warehouse system, this is a quick guide that will help you understand WMS Special Locations as well as how to manage them📋. There is a number of WMS Special Location Types that can be found in the system which are described below.

WMS Special Location Types 📌​

  • No Type - This is a none specified wms special location type and can be used to set the location to be unspecified by other types🚫.
  • Inbound - Inbound locations are created for pallets to be scanned into the system⬅️, the Inbound Only Type indicates that the selected location can only scan in pallets.

.

  • Outbound - The opposite to an Inbound type the Outbound type is used to take out a scanned out pallet➡️, when the lane location is set to the Outbound Only Type it is restricted to scanning out pallets.

.

  • Shuffle Position - This wms special location type allows the selected location to execute a pallet shuffle task and is very useful for multiple pallets🔁.
  • Reject Outlet - This specifies a special location where rejected pallets can be placed🚮.

Managing WMS Special Locations 🌍​

In order to create WMS Special Location, the user needs to navigate to the tool bar button found on the left side of the screen. This will open the tool bar menu.

.

Tool Bar 🔨​

Once the Tool Bar is open📂, find and click the location button to expand your options.

.

Location Menu 📄​

This menu allows user to customize specified locations🔧. Click the Special Location button to navigate to the Special Locations Panel🎯.

.

Special Location Panel 📋​

The Special Locations Panel is the main component used to manage special locations. It allows users to define their own special location by simply providing the required parameters✒️.

.

Manage Special Location 👔​

Users can also chose to Create, Update or Delete special locations by selecting their operation action in the Operation Type drop menu🛂.

.

Select Location ✨​

After having selected the Operation Type, double click the intended location in order to provide the location Id🆔.

.

Submit Special Location Request ✅​

Simply enter the required information and populate the selected location by clicking the plus button➕. Click Submit Request to complete the special location request✔️.

.

Special Location Request Completed 🔔​

Upon completion the user is notified of the request completion🎊!

.

Quick Special Location Request⚡​

A quick shortcut to a special location request is to select and then right click on a lane location🖱. Click the lane tab to extend your options.

.

1. Navigate To Quick Special Locations​

Once open, search and click the Quick Special Locations drop down menu to manage your selected location🔍.

.

2. Manage Quick Special Location​

This will provide the user with a shortcut to manage their special locations. Click on the desired special location type and you're done🎉!

.

3. Delete Quick Special Location​

Easily delete a special location by just finding and clicking the special location you would like to remove🗑️.

.

Congradulations⭐! You have just learnt how to manage your WMS Special Locations🎓!
